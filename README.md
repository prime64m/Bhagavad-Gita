# Bhagavad Gita - Divine Wisdom

A beautiful web application to explore the sacred verses of the Bhagavad Gita with detailed explanations and translations.

## Features

- **Random Verse**: Get a random shloka with Sanskrit text, transliteration, translation, and explanation
- **Browse Chapters**: Navigate through all 18 chapters of the Bhagavad Gita
- **Verse Selection**: Select specific verses from any chapter
- **Beautiful UI**: Immersive 3D background with modern, responsive design

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- React Three Fiber (for 3D background)

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd bhagavad-gita-3d-main

# Step 3: Install the necessary dependencies
npm install

# Step 4: Start the development server
npm run dev
```

The application will be available at `http://localhost:8080`

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build for production
- `npm run preview` - Preview the production build
- `npm run lint` - Run ESLint
